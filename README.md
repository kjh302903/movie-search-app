# movie-search-app

OMDb API를 활용한 영화 검색 사이트를 클론한 페이지입니다.<br/>
vercel 서비스를 이용하여 자동 배포 하였습니다.<br/>
React를 사용하지않고 SPA 구조를 VanilaJs를 사용하여 만들었습니다.</br />
[Link](https://movie-search-app-ten-cyan.vercel.app/#/)
